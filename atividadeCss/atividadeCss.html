CSS

Definição

O CSS (Cascading Style Sheets), é uma linguagem de marcação utilizada para descrever a apresentação visual de documentos HTML. Ela é fundamental para o design e a formatação de páginas da web, permitindo que os desenvolvedores controlem o layout, as cores, as fontes, o espaçamento e outros aspectos visuais de um site ou aplicativo web.

Como aplicar o CSS

Existem três formas principais de aplicar estilos CSS a elementos HTML: inline, incorporado e em folha de estilo externa.

Inline

Os estilos inline são aplicados diretamente a um elemento HTML usando o atributo style. Eles são especificados como valores de atributo e se aplicam apenas ao elemento em questão.

Exemplo:

  • <p style="color: blue; font-size: 16px;">

Incorporado

Os estilos incorporados são definidos dentro do próprio documento HTML, geralmente no cabeçalho (head) do documento, usando a tag <style>. Eles se aplicam a elementos específicos ou a toda a página.

Folha de Estilo Externa

Os estilos em folhas de estilo externas são definidos em arquivos CSS separados e, em seguida, vinculados ao documento HTML usando a tag <link> no cabeçalho.

Sintaxe de uma regra CSS

Uma regra CSS (Cascading Style Sheet) consiste em um conjunto de instruções que definem como os elementos HTML devem ser estilizados em uma página da web. A sintaxe básica de uma regra CSS é a seguinte: 

  • seletor {
    propriedade: valor;
}

Exemplo:

  • p {
    font-size: 16px;
    color: #333;
    margin-bottom: 20px;
    text-align: justify;
}

font-size: 16px;: Define o tamanho da fonte dos parágrafos como 16 pixels.
color: #333;: Define a cor do texto dos parágrafos como um cinza escuro (#333).
margin-bottom: 20px;: Adiciona uma margem inferior de 20 pixels aos parágrafos para espaçá-los verticalmente.
text-align: justify;: Alinha o texto à esquerda e justifica o espaçamento entre as palavras nos parágrafos.

Tipos de Seletores

  1. Seletor Universal: é representado pelo caractere asterisco (*) e seleciona todos os elementos HTML em uma página.
    2. * { margin: 0; padding: 0; }
  2. Seletor de Tipo: seleciona todos os elementos HTML de um tipo específico. Você pode usar o nome do elemento como o seletor e, quando aplicado, todas as regras de estilo serão aplicadas a todos os elementos desse tipo na página.
    3. p { color: blue; font-size: 16px; }
  3. Seletor de Classe: permite selecionar elementos HTML que têm um atributo de classe específico. Isso é útil quando você deseja aplicar estilos a um grupo de elementos que compartilham a mesma classe, independentemente do tipo de elemento.
    4. .nome-da-classe { propriedade: valor; }
  4. Seletor de ID: é usado para selecionar um elemento HTML que possui um atributo de ID específico. IDs são atributos exclusivos em uma página, o que significa que cada ID deve ser único dentro do documento HTML.

Angular

O Angular é um framework de desenvolvimento front-end, inicialmente criado pelo Google e amplamente utilizado na construção de aplicativos web complexos e interativos. Ele se destaca por sua arquitetura baseada em componentes, que permite dividir a interface do usuário em partes reutilizáveis.

O Angular é construído sobre o TypeScript, uma linguagem superset do JavaScript que traz tipagem estática e outros recursos avançados de programação orientada a objetos para o desenvolvimento web. Isso torna o código mais seguro, escalável e fácil de manter.

Além disso, o Angular oferece recursos como roteamento, que permite a criação de aplicativos de página única (SPA) com transições suaves entre páginas, e injeção de dependência, facilitando a gestão de componentes e serviços.

Outra característica notável do Angular é sua comunidade ativa e ecossistema robusto, que inclui uma ampla variedade de bibliotecas e ferramentas de terceiros para expandir suas funcionalidades.

Em resumo, o Angular é uma escolha popular para desenvolvedores que buscam construir aplicativos web avançados e escaláveis, graças à sua arquitetura de componentes, tipagem estática e recursos abrangentes.

Versões

O Angular, ao longo de sua evolução, teve várias versões principais. Aqui estão as versões do Angular:

AngularJS (1.x): Esta foi a primeira versão do Angular, lançada em 2010. Foi uma das primeiras tentativas de criar um framework front-end robusto para criar aplicativos web dinâmicos. O AngularJS é amplamente diferente das versões posteriores em termos de arquitetura e abordagem.

Angular 2+: Esta versão marcou uma reescrita completa do Angular, abandonando o AngularJS original. Começou com o Angular 2 em 2016 e, desde então, tem sido atualizado regularmente. Algumas versões notáveis incluem o Angular 4, Angular 5, Angular 6 e assim por diante. A versão "2+" foi chamada assim porque a equipe de desenvolvimento pretendia enfatizar a frequência das atualizações em vez de grandes mudanças de versão.

Angular 10 e posteriores: O Angular continuou a evoluir com melhorias de desempenho, recursos adicionais e correções de bugs. A numeração das versões continuou a aumentar, por exemplo, Angular 10, Angular 11, Angular 12 e assim por diante. Sendo a mais recente o Angular 16

Instalando Angular Cli

Para fazer isso, execute o seguinte comando no terminal ou prompt de comando:

npm install -g @angular/cli


Isso instalará o Angular CLI globalmente, tornando-o acessível em qualquer lugar do seu sistema.

Verificando a instalação

Para verificar se o Angular CLI foi instalado com sucesso, você pode executar o seguinte comando:

ng --version

Isso deverá exibir a versão do Angular CLI instalada.

Criando Projeto em Angular

Agora que o Angular CLI está instalado, você pode criar um novo projeto Angular usando o comando ng new. Substitua <nome-do-projeto> pelo nome que você deseja dar ao seu projeto:

ng new <nome-do-projeto>


Executando aplicações

Primeiramente para executar a aplicação você deve acessar o terminal ou prompt de comando no caminho da pasta do projeto em angular e rodar a seguinte linha de comando:

ng serve
